## 회원가입 사용자 플로우

### 진입점

```
로그인 화면 (/login)
└─ "회원 가입" 버튼 클릭 → /signup
```

---

### STEP 1 — 본인 인증 방법 선택 (`/signup`)

```
┌──────────────────────────────────────────────────────────┐
│  회원 가입을 위해 본인 인증을 진행해 주세요                    │
├──────────────────────────────────────────────────────────┤
│  1. 내 명의 휴대폰이 있어요   → NICE 본인인증 (외부 리다이렉트) │
│  2. 타인 명의 휴대폰이 있어요  → /signup/sms-verification   │
└──────────────────────────────────────────────────────────┘
```

#### 선택 1 — 내 명의 (NICE)

```
/signup/nice-return  (자동 처리, 스피너만 표시)
  └─ NICE 결과 수신 → 이름·휴대폰 자동 저장
  └─ → /signup/agreement  (STEP 2)
```

#### 선택 2 — 타인 명의 (SMS 인증)

```
/signup/sms-verification
  입력: 휴대폰 번호 (010-0000-0000 형식)
  버튼: "인증번호 받기"  → SMS 발송
  입력: 인증번호 6자리
        (Android Chrome: WebOTP 자동입력 / iOS Safari: 자동완성)
  버튼: "인증하기"  → 서버 검증
  └─ 성공 → /signup/agreement  (STEP 2)
```

---

### STEP 2 — 약관 동의 (`/signup/agreement`)

```
/signup/agreement
  표시: 개인정보 수집·이용 약관 (스크롤)
  버튼: "동의하기"
  └─ → /signup/domestic-foreign  (STEP 3)
```

---

### STEP 3 — 회원 유형 선택 (`/signup/domestic-foreign`)

```
┌─────────────────────────────────────┐
│  1-내국인  → /signup/signup-rrn     │
│  2-외국인  → /signup/signup-frn     │
└─────────────────────────────────────┘
```

---

### STEP 4-1 — 내국인 · 주민등록번호 (`/signup/signup-rrn`)

```
/signup/signup-rrn
  입력: 이름 (한글, 최대 4자)
  입력: 주민등록번호 앞 6자리 / 뒤 7자리
  표시: 휴대폰 번호 (읽기 전용)
  입력: 주소 (Kakao 주소 검색)
  버튼: "다음"  ← 모든 필드 입력 시 활성화
  └─ → /signup/set-password  (STEP 5)
```

---

### STEP 4-2 — 외국인 · 외국인등록번호 (`/signup/signup-frn`)

```
/signup/signup-frn
  입력: 한글 이름 (최대 4자)
  입력: 영문 이름 (선택)
  입력: 외국인등록번호 앞 6자리 / 뒤 7자리
  표시: 휴대폰 번호 (읽기 전용)
  입력: 주소 (Kakao 주소 검색)
  링크: "여권번호로 가입하기" (외국인등록번호 없는 경우)
  버튼: "다음"  ← 이름·등록번호·주소 입력 시 활성화
  └─ "다음"               → /signup/set-password  (STEP 5)
  └─ "여권번호로 가입하기"  → /signup/signup-pn    (STEP 4-2-1)
```

---

### STEP 4-2-1 — 외국인 · 여권번호 (`/signup/signup-pn`)

```
/signup/signup-pn
  입력: 한글 이름 (최대 4자)
  입력: 영문 이름 (선택)
  선택: 성별 (남성 / 여성)
  입력: 여권번호
  선택: 국적
  입력: 생년월일 (YYYY-MM-DD)
  표시: 휴대폰 번호 (읽기 전용)
  입력: 주소 (Kakao 주소 검색)
  버튼: "다음"  ← 이름·여권번호·국적·생년월일·주소 입력 시 활성화
  └─ → /signup/set-password  (STEP 5)
```

---

### STEP 5 — 비밀번호 설정 (`/signup/set-password`)

```
/signup/set-password
  입력: 비밀번호 (8–64자)
        실시간 조건 표시:
          ✓ 8자 이상   ✓ 영문 포함   ✓ 숫자 포함   ✓ 특수문자 포함
  입력: 비밀번호 확인
  버튼: "다음"  ← 두 항목 모두 유효할 때 활성화
        → 서버 회원가입 API 호출
  └─ 성공 → /signup/complete  (STEP 6)
```

---

### STEP 6 — 가입 완료 (`/signup/complete`)

```
/signup/complete
  표시: "회원 가입 완료"
  버튼: "로그인하기"
  └─ → /login
```

---

### 온보딩 플로우 (최초 로그인 후 자동 진입)

#### OB-1 — 시작 (`/onboarding`)

```
/onboarding
  표시: "안녕하세요, {이름}님! 회원 가입을 환영합니다."
  버튼: "시작하기"
  └─ → /onboarding/worker-type  (OB-2)
  헤더 뒤로가기: 로그아웃 확인 팝업
    └─ "나가기" → 로그아웃 → /login
```

#### OB-2 — 근무자 유형 선택 (`/onboarding/worker-type`)

```
┌──────────────────────────────────────────────────────────┐
│  1. 일반     → /onboarding/payroll-account       (OB-4)  │
│  2. 용역     → /onboarding/outsourcing           (OB-3-1) │
│  3. 장비기사 → /onboarding/engineer              (OB-3-2) │
└──────────────────────────────────────────────────────────┘
```

#### OB-3-1 — 용역회사 선택 (`/onboarding/outsourcing`)

```
/onboarding/outsourcing
  선택: 용역회사 드롭다운 (서버 목록)
  버튼: "다음"  ← 회사 선택 시 활성화
  └─ → /onboarding/payroll-account  (OB-4)
```

#### OB-3-2 — 장비기사 정보 (`/onboarding/engineer`)

```
/onboarding/engineer
  선택: 구분 (대표자 / 직원)
  입력: 대표자명 또는 소속 법인명
  버튼: "다음"  ← 이름 입력 시 활성화
  └─ → /onboarding/daily-wage  (OB-7)  ← 급여계좌 단계 건너뜀
```

#### OB-4 — 급여 지급 방식 (`/onboarding/payroll-account`)

```
┌──────────────────────────────────────────────────────────────┐
│  1. 본인 계좌로 지급   → /onboarding/my-account      (OB-5-1) │
│  2. 가족 계좌로 지급   → /onboarding/family-account  (OB-5-2) │
│  3. 소속 회사로 지급   → /onboarding/daily-wage      (OB-7)   │
│     (용역 근무자에게만 표시)                                     │
└──────────────────────────────────────────────────────────────┘
```

#### OB-5-1 — 본인 계좌 (`/onboarding/my-account`)

```
/onboarding/my-account
  선택: 은행명
  입력: 계좌번호 (- 없이 숫자)
  입력: 예금주명
  버튼: "저장"
  └─ → /onboarding/documents  (OB-6)
```

#### OB-5-2 — 가족 계좌 (`/onboarding/family-account`)

```
/onboarding/family-account
  입력: 예금주 이름
  선택: 관계 (배우자 / 부모 / 자녀 등)
  선택: 은행명
  입력: 계좌번호 (- 없이 숫자)
  버튼: "저장"
  └─ → /onboarding/documents  (OB-6)
```

#### OB-6 — 서류 등록 (`/onboarding/documents`)

```
/onboarding/documents
  표시: 필수 서류 목록 (근무자 유형·국적에 따라 다름)
        예: 신분증, 통장사본, 기초안전보건교육이수증 등
  각 서류: "파일 선택" → 파일 업로드
  버튼: "나중에 하기"  → /home  (홈 배너에서 나중에 재진입 가능)
  버튼: "다음"         → /onboarding/daily-wage  (OB-7)
```

#### OB-7 — 일급 입력 (`/onboarding/daily-wage`)

```
/onboarding/daily-wage
  입력: 일급(원)  예: 100,000
  버튼: "저장"  ← 금액 > 0 시 활성화
        → 온보딩 완료 API 호출
  └─ 완료 팝업 "확인" → /home
```

### 전체 경로 요약

| 경로 | 인증 | 국적 | 신분증 | 스크린 흐름 |
|------|------|------|--------|-----------|
| 1-내국인 | NICE 또는 SMS | 내국인 | 주민등록번호 | 회원가입 → 인증 → 약관 → 1-내국인 → 이름/주민번호/주소 → 비밀번호 → 완료 → 온보딩 |
| 2-외국인 | NICE 또는 SMS | 외국인 | 외국인등록번호 | 회원가입 → 인증 → 약관 → 2-외국인 → 한글이름/영문이름/등록번호/주소 → 비밀번호 → 완료 → 온보딩 |
| 2-외국인(여권) | NICE 또는 SMS | 외국인 | 여권번호 | 회원가입 → 인증 → 약관 → 2-외국인 → 여권번호로 가입하기 → 한글이름/영문이름/성별/여권번호/국적/생년월일/주소 → 비밀번호 → 완료 → 온보딩 |

온보딩 근무자 유형별 분기:

| 유형 | 온보딩 경로 |
|------|-----------|
| 일반 | 유형선택 → 급여방식 → (본인계좌 \| 가족계좌) → 서류등록 → 일급입력 → 홈 |
| 용역 | 유형선택 → 용역회사 → 급여방식 → (본인계좌 \| 가족계좌 \| 소속회사) → 서류등록 → 일급입력 → 홈 |
| 장비기사 | 유형선택 → 장비기사정보 → 일급입력 → 홈 |
